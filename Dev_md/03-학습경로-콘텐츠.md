# 학습 경로 콘텐츠 상세

## 공통 패턴

모든 학습 경로 페이지는 동일한 UI 패턴을 따름:

```
┌─────────────────────────────────────────┐
│ Navbar                                  │
├──────────┬──────────────────────────────┤
│ Sidebar  │ Content Area                 │
│ (목차)   │                              │
│          │ ┌────────────────────────┐   │
│ > 섹션1  │ │ Section Header         │   │
│   섹션2  │ │ h1 + description       │   │
│   섹션3  │ ├────────────────────────┤   │
│   섹션4  │ │ Section Content        │   │
│   섹션5  │ │ h2, h3, p, ul, ol      │   │
│   섹션6  │ │ TipBox, ComparisonTable│   │
│          │ ├────────────────────────┤   │
│          │ │ ← 이전  |  다음 →      │   │
│          │ └────────────────────────┘   │
├──────────┴──────────────────────────────┤
│ Footer                                  │
└─────────────────────────────────────────┘
```

### 컴포넌트 구조

```tsx
// SECTIONS 배열 정의
const SECTIONS = [
  { id: 'section-id', icon: 'fa-icon', ko: '한글명', en: 'English Name' },
  // ...
];

// 메인 컴포넌트
export default function PageName(): ReactElement {
  const { language } = useLanguage();
  const [activeSection, setActiveSection] = useState('first-section');
  const isKo = language === 'ko';
  // sidebar nav + conditional section rendering + prev/next nav
}

// 개별 섹션 함수
function SectionName({ isKo }: { isKo: boolean }): ReactElement {
  return (
    <div className="guide-content-header">...</div>
    <div className="guide-section">...</div>
  );
}
```

### 공통 사용 컴포넌트
- `SEOHead` — 페이지별 SEO 메타태그
- `TipBox` — 팁(tip), 중요(important), 경고(warning), 위험(danger) 박스
- `useLanguage()` — 한/영 전환 컨텍스트
- CSS: `guide-page`, `guide-layout`, `guide-sidebar`, `guide-content`, `guide-section`, `guide-section-nav`
