# 아키텍처 설계

## 앱 구조

```
main.tsx
  └── App.tsx (BrowserRouter)
        └── ThemeProvider
              └── LanguageProvider
                    └── ToastProvider
                          └── AuthProvider
                                └── CartProvider
                                      └── PublicLayout
                                            ├── Navbar
                                            ├── <Routes> (Lazy loaded)
                                            └── Footer
```

## 라우팅 구조

| 경로 | 컴포넌트 | 설명 |
|------|----------|------|
| `/` | Home | 메인 홈페이지 |
| `/login` | Login | 로그인 |
| `/register` | Register | 회원가입 |
| `/forgot-password` | ForgotPassword | 비밀번호 찾기 |
| `/mypage` | MyPage | 마이페이지 |

| `/sampling` | Sampling | 표본추출 |
| `/measurement` | Measurement | 측정과 척도 |
| `/survey` | Survey | 설문조사법 |
| `/experiment` | Experiment | 실험연구 |
| `/qualitative` | Qualitative | 질적연구 |
| `/statistics` | Statistics | 통계분석 |

모든 페이지는 `React.lazy()` + `Suspense`로 코드 분할.

## Context 구조

### ThemeContext
- 다크모드 토글 (`light` / `dark`)
- 컬러 테마 선택 (primary color)
- `localStorage` 저장

### LanguageContext
- 한/영 전환 (`ko` / `en`)
- `localStorage` 저장
- 전역 `language` 상태 제공

### AuthContext
- Supabase Auth 연동
- 로그인/로그아웃/회원가입
- 세션 자동 갱신
- `user` 상태 전역 제공

### ToastContext
- 성공/에러/정보/경고 토스트
- 자동 사라짐 타이머
- 스택형 다중 표시

### CartContext
- 장바구니 아이템 관리
- `localStorage` 저장
- 수량 변경/삭제/전체 비우기

## 스타일 아키텍처

```
index.css (import hub)
  ├── styles/base.css          # CSS 변수, 기본 레이아웃
  ├── styles/navbar.css        # Navbar 고정/투명/스크롤
  ├── styles/hero.css          # 히어로 그라데이션/애니메이션
  ├── styles/home.css          # 홈페이지 그리드/카드
  ├── styles/guide-pages.css   # 사이드바+콘텐츠 레이아웃
  ├── styles/code-blocks.css   # 코드 하이라이팅
  ├── styles/footer.css        # 푸터 다크 배경
  ├── styles/community.css     # 커뮤니티 스타일
  ├── styles/auth.css          # 인증 폼 스타일
  ├── styles/toast.css         # 토스트 알림
  ├── styles/dark-mode.css     # html[data-theme="dark"] 오버라이드
  ├── styles/animations.css    # @keyframes 정의
  └── styles/responsive.css    # 미디어 쿼리 (1280/1024/768/480/360px)
```

### 반응형 브레이크포인트

| 브레이크포인트 | 주요 변경 |
|---------------|----------|
| 1280px | 컨테이너 패딩 축소, 그리드 2열 |
| 1024px | 모바일 메뉴 활성화, 가이드 사이드바 축소 |
| 768px | 단일 열 레이아웃, 사이드바 수평 전환 |
| 480px | 폰트 축소, 패딩 최소화 |
| 360px | 최소 모바일 대응 |

## 데이터 흐름

```
Supabase (Backend)
  ├── Auth → AuthContext → 모든 컴포넌트
  ├── Database (rs_ prefix) → storage utils → 페이지/컴포넌트
  └── Storage → ImageUpload / Avatar

PortOne (Payment)
  └── portone.ts → Checkout → OrderConfirmation

localStorage
  ├── theme, colorTheme → ThemeContext
  ├── language → LanguageContext
  └── cart items → CartContext
```

## 배포 파이프라인

```
Push to main branch
  → GitHub Actions trigger
    → actions/checkout@v4
    → actions/setup-node@v4 (Node 20)
    → npm ci
    → npm run build (with Supabase/PortOne secrets)
    → actions/configure-pages@v4
    → actions/upload-pages-artifact@v3
    → actions/deploy-pages@v4
      → research.dreamitbiz.com (GitHub Pages)
```
