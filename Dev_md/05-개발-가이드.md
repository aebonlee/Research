# 개발 가이드

## 환경 설정

### 필수 조건
- Node.js 20+
- npm 10+
- Git

### 설치 및 실행

```bash
# 의존성 설치
cd D:\dreamit-web\Research
npm install

# 개발 서버 실행 (포트 5182)
npm run dev

# 타입 체크
npm run typecheck

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

```env

VITE_PG_PROVIDER=your-pg-provider
```

## 코딩 컨벤션

### TypeScript
- 모든 컴포넌트에 `ReactElement` 반환 타입 명시
- Props 타입은 인라인 또는 별도 interface
- `type` import 사용 (`import type { ReactElement }`)

### 컴포넌트
- 함수 선언문 (`function` keyword) 사용
- `export default` 직접 사용
- 컴포넌트 파일명: PascalCase
- 한 파일에 메인 컴포넌트 + 섹션 함수 함께 배치

### CSS
- CSS 변수 기반 테마 시스템
- BEM-like 클래스명 (`.guide-nav-link`, `.guide-content-header`)
- 다크모드는 `html[data-theme="dark"]` 선택자
- 반응형은 `responsive.css`에 집중

### 이중 언어
- `useLanguage()` 훅으로 현재 언어 확인
- `isKo` boolean으로 한/영 분기
- 긴 텍스트는 JSX 내에서 직접 삼항 연산자 사용
- 짧은 레이블은 `translations.ts` 참조

### 환경변수 미설정
- `.env` 파일이 프로젝트 루트에 있는지 확인
- `VITE_` 접두사 필수

### GitHub Pages 배포 실패
- Repository Settings → Pages → Source: GitHub Actions
- Secrets 설정 확인 (VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY 등)
- CNAME 파일이 `public/` 디렉토리에 있는지 확인
