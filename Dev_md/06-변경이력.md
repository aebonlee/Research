# 변경 이력 (Changelog)

## v1.0.0 — 2026-04-09 (초기 릴리스)

### 프로젝트 초기화
- React 19 + Vite 7 + TypeScript 프로젝트 생성
- `study.dreamitbiz.com` 기반 아키텍처 적용
- package.json: `dreamitbiz-research` v1.0.0
- Vite 개발 서버 포트: 5182
- 기본 테마 색상: #1E40AF (Deep Blue)

### 설정 파일
- `src/config/site.ts` — 사이트 설정 (id: 'research', dbPrefix: 'rs_') + 8개 LEARNING_PATHS 정의
- `src/config/admin.ts` — 관리자 이메일 설정
- `vite.config.ts` — Vite 7 빌드 설정 (port 5182)
- `index.html` — 조사방법론 메타태그, OG/Twitter 카드
- `.env.example` — 환경변수 템플릿
- `public/CNAME` — research.dreamitbiz.com
- `public/robots.txt`, `public/sitemap.xml` — SEO 파일

### Context 시스템 (5개)
- `ThemeContext.tsx` — 다크모드 + 컬러 테마 관리
- `LanguageContext.tsx` — 한/영 언어 전환
- `AuthContext.tsx` — Supabase 인증 상태
- `ToastContext.tsx` — 알림 토스트 메시지
- `CartContext.tsx` — 장바구니 상태 관리

### 공유 컴포넌트 (13개)
- `Navbar.tsx` — 조사방법론 8개 학습경로 내비게이션 메뉴

- `AuthGuard.tsx` — 인증 보호 래퍼

- `CodeBlock.tsx` — 코드 블록 하이라이팅
- `HeroBackground.tsx` — 히어로 섹션 배경 애니메이션
- `HeroCarousel.tsx` — 히어로 캐러셀 슬라이더

- `Pagination.tsx` — 페이지네이션
- `ImageUpload.tsx` — 이미지 업로드
- `CommentSection.tsx` — 댓글 섹션

### 유틸리티 (8개)
- `supabase.ts` — Supabase 클라이언트 초기화
- `auth.ts` — 인증 헬퍼 함수
- `portone.ts` — PortOne 결제 연동

- `commentStorage.ts` — 댓글 데이터 CRUD

### 커스텀 훅 (2개)

- `useCountUp.ts` — 숫자 카운트업 애니메이션

### 스타일 (13개 CSS)
- `base.css` — CSS 변수 (#1E40AF 기본), 레이아웃
- `navbar.css` — 내비바 고정/투명/스크롤 효과
- `hero.css` — 히어로 그라데이션/파티클
- `home.css` — 홈페이지 그리드/카드/통계
- `guide-pages.css` — 사이드바+콘텐츠 가이드 레이아웃
- `code-blocks.css` — 코드 블록 테마
- `footer.css` — 푸터 다크 배경
- `community.css` — 커뮤니티 섹션

- `ResearchDesign.tsx` — 연구설계: 기초/문제·가설/유형/인과관계/모형/계획서
- `Sampling.tsx` — 표본추출: 모집단/확률/비확률/크기/오차/실전
- `Measurement.tsx` — 측정과 척도: 개념/수준/척도/신뢰도/타당도/개발
- `Survey.tsx` — 설문조사법: 개요/설계/질문/바이어스/방법/실전
- `Experiment.tsx` — 실험연구: 기초/설계/타당도/통제/준실험/사례
- `Qualitative.tsx` — 질적연구: 이해/면접/포커스그룹/관찰/근거이론/분석
- `Statistics.tsx` — 통계분석: 기술/추론/가설검정/t-ANOVA/회귀/비모수
